# Lexica - Quick Start Guide

## 🚀 Get Running in 30 Seconds

```bash
npm install
npm run dev
# Open http://localhost:3000
```

## 📖 First Time Use

1. Press `Esc` to open settings
2. Click "Choose Markdown File"
3. Upload any `.md` or `.txt` file
4. Press `Space` to start reading

## ⌨️ Essential Shortcuts

- `Space` / `→` : Next word
- `←` : Previous word  
- `Esc` : Settings/UI
- Mouse move: Show UI chrome

## 🎯 Quick Tips

- **Font size**: 18-20px works for most people
- **Window radius**: Start with 2, adjust as needed
- **Dark theme**: Recommended (default)
- **Full screen**: Press F11 for immersive reading

## 🎉 What Makes Lexica Special

Lexica is a **focus scaffolding system** that helps you read with:

- **Stable focal point** - No eye scanning
- **Peripheral context** - See surrounding words
- **Manual pacing** - You control speed
- **Structural awareness** - Always know where you are

Perfect for readers with ADHD, students, and knowledge workers.
